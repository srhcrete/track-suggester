CSS/Javascript Web Design Track

CSS is used for designing the presentation of Web pages. With CSS you can manipulate the colors, layouts, and fonts of a page. CSS also allows us to adapt the presenation of the page to various devices.

Javascript is the most common landuage for web and mobile development because it is the only language understood by all web browsers. It is used to add interactivity and animations to web pages.

Web Design is the process of determining the layout, colors, text styles, structure, graphics, images, and use of interactive features on a web page to make a positive and interactive user experience.

C#/.NET Track

C# is popular amoung larger established businesses and major corporations. It is usually used for building internal software and it is one of the most in-demand languages in the job market. If you like the idea of working for larger companies like, Microsoft, then C# is a great place to start.

.NET is a software framework developed by Microsoft, used to create applications that run on various devices. It compiles and executes programs written in the C# and Visual Basic languages.

Ruby/Rails Track

Ruby is popular amoung developers building interactive web applications. Such apps involve creating accounts, entering information, and interacting with dynamic content. Ruby has elegant syntax that is easy to read and write.

Rails is a framework written in Ruby and simplifies many of the common tasks associated with building web applications. Ruby on Rails is most popular amoung startups and smaller companies who are looking to build their product quickly.
